# Endorsed candidate photos

Drop campaign photos here named by candidate slug:

- `jeff-givan.jpg` — Jeff Givan, OH House District 78 (2026)

File naming
- Use lowercase, hyphen-separated: `first-last.jpg`
- Prefer JPG, ~1080px wide, ~3:2 or 4:5 portrait crop.
- The endorsements page (`/endorsements`) crops to a 52x52 square avatar
  with `object-position: center top`, so headshots should keep the
  subject's face in the upper third.

These paths are referenced from `ENDORSEMENT_OVERRIDES` in
`endorsements/index.html`.
